# Hanauta Lab. お仕事タイプ診断

就労継続支援B型事業所 Hanauta Lab.の利用予定者向け、作業適性診断ウェブアプリ。

---

## 概要

13問の質問に答えると、8タイプのうちあなたに合う「お仕事タイプ」が分かる診断テスト。完全静的サイトでサーバー不要、GitHub Pagesで無料公開できます。

---

## 特徴

- **完全スマホ対応** - レスポンシブデザイン
- **2〜3分で完結** - 13問のシンプル設計
- **8タイプ × 4作業** - Hanauta Lab.の実作業に紐づく具体的な提案
- **LINE誘導** - 結果ページから公式LINEへ直リンク
- **SNSシェア対応** - X、LINE、テキストコピー
- **シェアリンクで結果再表示** - URLハッシュで他人の結果も見られる
- **完全無料** - GitHub Pagesで0円運用

## 編集ガイド

### 質問を変更したい

`js/questions.js` を編集。各質問は以下の構造：

```javascript
{
  id: 1,
  axis: 'tempo',  // 'tempo' or 'connection' or 'joy'
  text: '質問文',
  options: [
    { text: '選択肢A', value: 'odayaka' },
    { text: '選択肢B', value: 'wakuwaku' }
  ]
}
```

### タイプ説明を変更したい

`js/types.js` を編集。各タイプの `description`, `works`, `message` を書き換え。

### 色やデザインを変更したい

`css/style.css` の `:root` セクションでカラー変数を変更。

```css
:root {
  --color-primary: #7B9B7E;  /* メインカラー */
  --color-accent: #D4A574;   /* アクセントカラー */
}
```

### LINEのURLを変更したい

`js/script.js` の2行目を変更。

```javascript
const LINE_URL = 'https://lin.ee/RCXr7yI';
```

### 公開URLを変更したい

`js/script.js` の4行目と `index.html` のOGPタグを更新。
